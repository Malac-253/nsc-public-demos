Most common fix

In Render → Environment, set AWS_S3_REGION_NAME to the exact bucket region (S3 console → your bucket → Properties → AWS Region, e.g. us-east-1 not “US East”). IAM user needs at least: 

s3:ListBucket on arn:aws:s3:::YOUR-BUCKET
s3:GetObject, s3:PutObject, s3:DeleteObject on arn:aws:s3:::YOUR-BUCKET/*
